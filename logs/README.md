# Logs Directory

This directory contains application logs generated by Winston.

## Log Files

- `error.log` - Contains only error-level logs
- `combined.log` - Contains all logs (info, warn, error, debug)

## Log Rotation

Logs are automatically rotated when they reach 5MB in size, keeping the last 5 files.

## Important

**Do not commit log files to version control!**

The `.gitignore` file should include:
```
logs/*.log
```
